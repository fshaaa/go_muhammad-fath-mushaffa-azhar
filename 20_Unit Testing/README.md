# Summary 20: Unit Testing
## Unit Testing
Unit Testing adalah pengujian pada perangkat lunak yang menguji setiap
unit atau komponen dari perangkat lunak.

## Tujuan Unit Testing
- Preventing Regression
- Confidence Level in Refactoring
- Improve Code Design
- Code Documentation
- Scaling the Team

## Level of Testing
- UI Testing
- Integration Testing
- Unit Testing
