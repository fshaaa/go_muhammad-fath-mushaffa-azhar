# Summary 19: Middleware
## Middleware
Middleware adalah suatu software layer yang berada di antara router dengan controller. 
Bertugas untuk mengecek apakah client sudah memiliki hak akses suatu API atau belum

## Echo Middleware
### Echo Pre
- HTTPSRedirect
- HTTPSWWWRedirect
- WWWRedirect
- AddTrailingSlash
- RemoveTrailingSlash
- MethodOverride
- Rewrite

- Logger 
- Gzip
